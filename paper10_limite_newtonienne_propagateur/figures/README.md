# Figures — BuP Paper 10

Ce dossier contient les figures finales sélectionnées pour le README principal et le fichier LaTeX `paper/main_fr.tex`.

Les figures brutes complètes restent dans les sous-dossiers `results/`.

## Figures retenues

| Fichier | Rôle |
|---|---|
| `fig_seed0_main_diagnostics.pdf` | Diagnostics complets du run principal : distribution de \(\alpha\), scatter \(\alpha\) vs \(R^2\), courbe \(d_s(\tau)\). |
| `fig_seed1_main_diagnostics.pdf` | Même diagnostic pour une deuxième réalisation indépendante de la matrice MI. |
| `fig_alpha_median_robustness.png` | Comparaison des médianes \(\alpha_{\rm med}\) entre runs de robustesse. |
| `fig_window_comparison_summary.png` | Comparaison des fenêtres \([0.10,0.40]\), \([0.15,0.50]\), \([0.20,0.60]\). |
| `paper10_summary_for_figures.csv` | Données utilisées pour générer les figures synthétiques. |
